## Инструкция по развёртыванию:

### Вариант 1: Через браузер (рекомендуется)

1. Загрузите файлы на сервер через SFTP/файловый менеджер:
   - `corporate-meals.php` → `wp-content/themes/gustolocal/patterns/`
   - `functions.php` → `wp-content/themes/gustolocal/` (заменить)
   - `setup-corporate-meals.php` → корень WordPress

2. Откройте в браузере: `https://gustolocal.es/setup-corporate-meals.php`

3. Скрипт автоматически:
   - Проверит наличие паттерна
   - Создаст страницу со слагом `corporate-meals`
   - Применит паттерн к странице
   - Оставит страницу как черновик для проверки

4. После проверки удалите `setup-corporate-meals.php` из соображений безопасности

### Вариант 2: Через WordPress CLI

Если у вас есть доступ к серверу через SSH:

```bash
# Загрузите файлы на сервер, затем:
cd /path/to/wordpress
wp eval-file setup-corporate-meals.php
```

## Проверка:

После выполнения скрипта:

1. Откройте админку WordPress: `https://gustolocal.es/wp-admin`
2. Перейдите в "Страницы" → найдите "Горячие обеды для офисов и детских садов"
3. Страница должна быть в статусе "Черновик"
4. Откройте для редактирования и проверьте контент
5. Замените placeholder-изображения на реальные фото
6. После проверки опубликуйте страницу

## Структура страницы:

- **Блок 1 (Hero)**: Две колонки - текст слева, фото справа
- **Блок 2**: "Для кого и как это работает" - офисы и детские сады
- **Блок 3**: "Выберите, как вам удобнее" - Glovo или напрямую
- **Блок 4**: Контакты (WhatsApp, Telegram)

## Примечания:

- Все placeholder-изображения нужно заменить на реальные фото
- Ссылки на Glovo и контакты уже настроены
- Страница создаётся как черновик для проверки
- После публикации страница будет доступна по адресу: `https://gustolocal.es/corporate-meals/`
