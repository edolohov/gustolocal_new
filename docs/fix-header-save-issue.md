# Решение проблемы зависания при сохранении хедера

## Возможные причины:

1. **Конфликт плагинов** - некоторые плагины могут блокировать сохранение template parts
2. **Проблемы с JavaScript** - ошибки в консоли браузера
3. **Слишком большой файл** - но наш файл небольшой (46 строк)
4. **Проблемы с правами доступа** - но права установлены правильно (644)

## Решения:

### 1. Проверьте консоль браузера
- Откройте DevTools (F12)
- Перейдите на вкладку Console
- Попробуйте сохранить хедер
- Посмотрите, есть ли ошибки JavaScript

### 2. Отключите плагины временно
- Отключите все плагины кроме WooCommerce
- Попробуйте сохранить хедер
- Если работает - включайте плагины по одному

### 3. Попробуйте через WP-CLI
Если админка зависает, можно обновить хедер через командную строку:
```bash
wp post update <ID> --post_content="<содержимое>"
```

### 4. Проверьте размер файла
- Файл должен быть небольшим (< 10KB)
- Наш файл: 2596 байт - нормально

### 5. Упростите хедер
Если проблема сохраняется, можно упростить структуру блоков.
